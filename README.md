# HR Analytics Dashboard — Power BI

## Project Overview
Interactive HR analytics dashboard built using Power BI connected to a SQL Server database. The dashboard enables analysis of workforce distribution, salary structure, and employee performance with drill-through capability.

## Business Problem
HR teams need a clear way to analyze employees, salaries, and performance across departments. This dashboard provides an interactive solution for data-driven HR decisions.

## Key Features
- SQL Server connection
- Data cleaning with Power Query
- Data modeling and relationships
- KPI cards for employees, salary, performance
- Department comparison
- Employee drill-through analysis
- Slicers, tooltips, and page navigation
